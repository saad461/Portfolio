## 📝 Description

A professional, high-performance personal portfolio website designed to showcase developer projects and technical expertise. Built with Next.js 14 and Tailwind CSS, it features a sleek modern dark theme with smooth scroll animations and a fully responsive layout. This project serves as a comprehensive digital resume, highlighting featured work and core competencies in web development and AI integration.

## ✨ Features

- **📱 Responsive Modern Design**: Optimized for a seamless experience across all devices, from mobile phones to large desktop monitors.
- **🌙 Professional Dark Theme**: Uses a curated color palette (#0F172A) with professional blue and cyan accents for a modern look.
- **🎬 Scroll-Triggered Animations**: Engaging fade-in and reveal effects powered by the Intersection Observer API.
- **💻 Featured Projects Showcase**: Detailed presentation of work including tech stacks, project descriptions, and direct links to live demos/source code.
- **📊 Categorized Skills**: Clearly organized technical proficiency sections covering Frontend, Backend, AI, and Development Tools.
- **🔍 SEO Optimized**: Integrated with Next.js Metadata API and OpenGraph tags for enhanced visibility and social media sharing.
- **⚡ Optimized Performance**: Built on the Next.js 14 App Router for lightning-fast loading speeds and efficient client-side transitions.
- **🛠️ Type-Safe Codebase**: Developed using TypeScript to ensure code quality, maintainability, and a robust developer experience.

## 🛠️ Tech Stack

| Technology | Purpose |
| :--- | :--- |
| **Next.js 14** | React framework for Server-Side Rendering (SSR) and efficient routing via App Router. |

## 🏁 Getting Started

### Prerequisites

- **Node.js**: version 18.17.0 or higher
- **npm** or **yarn**
- **Git**

### Installation Steps

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/saad461/personal-portfolio.git
    ```

2.  **Navigate to the project folder:**
    ```bash
    cd personal-portfolio
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Set up environment variables:**
    Create a `.env.local` file in the root directory (see [Environment Variables](#-environment-variables) section).

5.  **Run the development server:**
    ```bash
    npm run dev
    ```

6.  **Open the application:**
    Navigate to [http://localhost:3000](http://localhost:3000) in your browser.

## 🔐 Environment Variables

No sensitive environment variables are strictly required for the basic portfolio to run. However, you can configure the following in your `.env.local` for future-proofing or specific features:

| Variable | Placeholder | Description |
| :--- | :--- | :--- |
| `NEXT_PUBLIC_SITE_URL` | `https://your-domain.com` | The base URL of your deployed site for SEO and OpenGraph tags. |
